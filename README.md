# CryptoTime开发者社区官网

AI驱动的华语区Web3开发者社区 | 用AI降低Web3门槛

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **图标**: Lucide React
- **部署**: Vercel (推荐)

## 快速开始

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 项目结构

```
├── app/                    # Next.js App Router
│   ├── globals.css        # 全局样式
│   ├── layout.tsx         # 根布局
│   └── page.tsx           # 首页
├── components/            # React组件
│   ├── Navigation.tsx     # 导航栏
│   └── HeroSection.tsx   # Hero部分
├── public/               # 静态资源
└── ...配置文件
```

## 设计特色

- 🌌 **深色宇宙主题**: 参考现代Web3设计风格
- 🎨 **渐变背景**: 从中心橙黄色到边缘深色的径向渐变
- 🪐 **动态元素**: 浮动星球和轨道线动画
- ✨ **发光效果**: 中央Logo的黄金发光效果
- 📱 **响应式设计**: 完美适配各种设备尺寸

## 开发说明

- 使用TypeScript确保类型安全
- 采用Tailwind CSS进行样式管理
- 组件化开发，便于维护和扩展
- 支持深色主题和现代UI设计
